# Overview

## ACTIVSg70k: 70,000 bus synthetic grid on footprint of eastern United States

This ACTIVSg70k test case is built on the eastern United States. This case covers 52 areas. It bears no relation to the actual grid in this location, except that generation and load profiles are similar, based on public data. The case is provided in PowerWorld format, including binary (PWB/PWD), PowerWorld auxiliary text format, Matpower format, PSS/E raw format, and PSLF epc format.
